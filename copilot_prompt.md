# Copilot Project Prompt – HTX Analytics

Ты — мой помощник-разработчик. Мы строим **аналитический интерфейс для биржи HTX**, без функций торговли. Проект предназначен **только для индивидуального использования** на моём ПК (Windows 11 с WSL2). Главная цель — создать удобную систему аналитики портфеля и инструментов для исследования данных через ИИ.

## Основные задачи проекта

1. **Фронтенд (Next.js + shadcn/ui)**
   - Лёгкий интерфейс для анализа портфеля.
   - Отображение данных через **API HTX** и через **CSV-экспорт**, чтобы учесть монеты, которые нельзя подтянуть по API.
   - Страницы: `dashboard`, `portfolio`, `analytics`.
   - Поддержка графиков (charts), таблиц, простых фильтров.
   - Всё должно быть визуально чисто и понятно — не перегружено.

2. **ИИ-аналитика (Docker + FinGPT + GCP)**
   - Запуск модели **FinGPT** в Docker.
   - Интеграция с Google Cloud Platform (GCP).
   - Автоматическая настройка инфраструктуры через **Terraform**.
   - Возможность дообучения модели на моих данных (портфель, сделки).
   - Хранение конфиденциальных данных (ключей, моделей) через **Google Secret Manager**.

3. **Интеграция двух частей**
   - Интерфейс должен уметь отправлять данные в ИИ-модуль и получать обратно аналитику.
   - Общий воркфлоу:
     1. Я загружаю CSV или подключаю API HTX.
     2. Интерфейс показывает портфель.
     3. Я отправляю данные в FinGPT для аналитики.
     4. Результаты отображаются обратно в UI.

## Технические требования

- **ОС:** Windows 11, работа через **WSL2**.
- **Локальная разработка:** Qoder для генерации кода и MCP-серверы для задач (CSV, Docker, Terraform).
- **Тестирование:** GitHub Codespaces + Copilot для генерации тестов.
- **CI/CD:** GitHub Actions для запуска тестов при push/PR.
- **Секреты:**
  - HTX API-ключ шифруется локально через **Fernet (Python cryptography)**.
  - GCP ключи и данные проекта хранятся в **Google Secret Manager**.
  - Codespaces использует **GitHub Codespaces Secrets**.
